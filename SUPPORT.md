# Support

## Before requesting help

1. Read [Troubleshooting](docs/TROUBLESHOOTING.md).
2. Run `NO_COLOR=1 codex-hud`.
3. Run `python3 -m unittest discover -s tests -v` from a repository checkout.
4. Confirm the operating system and versions of Python, Codex, tmux and OMX.

## Where to ask

- Use a bug report for reproducible defects.
- Use a feature request for proposed behavior.
- Use private vulnerability reporting for security issues.

Never attach raw session logs, credential files or unredacted environment dumps. Replace usernames, hostnames, addresses, paths, account identifiers and tokens with placeholders.
